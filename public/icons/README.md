# PWA Icons

Icon untuk Progressive Web App (PWA).

## Generate Icons

Untuk regenerate icon dari SVG, jalankan:

```bash
node public/icons/generate-icons.js
```

Ini akan menghasilkan icon dalam berbagai ukuran:

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Cara Install PWA di HP

### Android (Chrome)

1. Buka aplikasi di browser Chrome
2. Tap menu (3 titik) di pojok kanan atas
3. Pilih "Install app" atau "Add to Home Screen"
4. Konfirmasi install

### iOS (Safari)

1. Buka aplikasi di browser Safari
2. Tap tombol Share (kotak dengan panah)
3. Scroll ke bawah dan pilih "Add to Home Screen"
4. Edit nama jika perlu, lalu tap "Add"

### Desktop (Chrome/Edge)

1. Buka aplikasi di browser
2. Klik ikon install (+) di address bar
3. Atau buka menu (3 titik) > "Install [App Name]"
4. Konfirmasi install

Setelah diinstall, aplikasi akan muncul seperti aplikasi native dan bisa dibuka tanpa browser.
